What I'm good at?

Python & Pandas

Almost all of my projects have incorporated python in some way. Many of my projects involve the use of pandas to enhance data cleaning and analysis. I also used jupyter notebook to easily display dataframes and visualizations from the projects. Other libraries include matplotlib and plotly express.

Tableau

I have used Tableau Public to create visualizations to tell stories about data. From the interactive visualizations, I was able to analyze the data by applying different filters to the visualizations. Created tableau stories from worksheets and dashboards.

R

In R, I have performed statistical analyses on data. Using tidyverse and its ggplot2 library, I have cleaned datasets and displayed visualizations such as heatmaps and boxplots. Performed hypothesis tests through t-tests, anova tests, and chi-squared tests. Created multiple linear regression models to evaluate data.

SQL

I used PostgreSQL via pgAdmin to store and transform data. To organize tables, I have created ERD's to map out the variables and relationships for the different tables. Imported data into tables and used different joins to export the data for further analysis.
